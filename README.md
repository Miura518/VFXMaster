# VFXMaster
[![license](https://img.shields.io/badge/LICENSE-MIT-green.svg)](LICENSE)  
VFXMasterはUnityでエフェクト制作を行うために制作された多機能シェーダーです。  

##要件  
* 動作環境:Unity 6000.060f1  
* レンダーパイプライン:UniversalRenderPipeline

##インストール
* Window > PackageManagerからPackageManagerを開く
* 「+」ボタン > Install package from git URL
* 以下のURLを入力  
```
https://github.com/Miura518/VFXMaster.git?path=Assets/VFXMaster
```
